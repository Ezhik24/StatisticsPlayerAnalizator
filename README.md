<H1>Программа,с помощью которой вы можете получить топ 10 игроков по статистике,которую введете</H1>

<H2>Чтобы запустить программу,сделайте следующие действия</H2>

<H3>Для линукс:</H3>
1. <code>python3 -m venv .venv</code> <BR>
2. <code>source .venv/bin/activate</code> <BR>
3. <code>pip install pyyaml</code> <BR>
4. <code>.venv/bin/python3 main.py</code> <BR>
<H3>Для Windows</H3>
1. <code>python3.11.exe -m venv .venv</code> <BR>
2. <code>.venv/Scripts/pip3.11.exe install pyyaml</code> <BR>
3. <code>.venv/Scripts/python3.11.exe main.py</code>
